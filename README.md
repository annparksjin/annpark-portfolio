# annpark-portfolio

Personal portfolio website for Ann Park.

## Overview

This project is a static portfolio site that showcases:

- About / profile information
- Video project previews
- Interactive projects
- User manual style personal profile
- Additional project experience
- Contact links

## Stack

- HTML
- CSS
- Vanilla JavaScript

## Running Locally

Open [index.html](/Users/annpark/Documents/New%20project/index.html) in a browser.

If you want a local server, you can run:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
